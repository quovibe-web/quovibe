## Features

### Portfolio Import Hub
- New **`/p/:pid/import`** hub page with a tile grid covering all import flows: trades CSV, prices CSV, PP-XML, and `.db` restore.
- Dashboard empty-state import button and the trades-wizard "prices CSV" shortcut both route to the hub.
- Portfolio-switch now lands on the hub instead of the CSV wizard directly.

### Price CSV Wizard
- New **4-step `/p/:pid/import/prices`** wizard: security picker → upload + pre-POST sniff → column mapping (close, date, volume, OHLC) → confirm/execute.
- Security picker filter helper with name + ISIN/ticker search.
- Reuses shared sniff helpers and type-column key utilities from the trades wizard.

### Dashboard
- **Pencil affordance** on widget titles for inline rename — click the title to enter edit mode.

## Fixes

- **BUG-181** — removed annualized time-series chart mode; cumulative-only matches PP parity.
- `preserveCustomTables` implemented in import service — `vf_exchange_rate` rows now copy from the live DB into a freshly re-imported DB, so FX rates are preserved across PP-XML re-imports.

## i18n

- Terminology pass across all 8 languages with internal-consistency vitest coverage.
- New keys: `navigation.items.import`, `csv-import.hub.*`, `csv-import.priceWizard.*`, `transactions.actions.importCsv`.
